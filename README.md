# Connect-with-ugi

Steps To Run Project:

1) Install Node
2) Install MongoDb

//Clone Repository
3) Clone this repository 

4) cd Connect-with-ugi

//backend setup
5) cd backend
5.1)Open .env file and set your MongoDb URL
5.2) npm install
5.3) npm start

//frontend setup
6) cd frontend
6.1) npm install
6.2) npm start
